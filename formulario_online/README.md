# Plataforma de Formulários No-Code

Aplicação desenvolvida no Bubble.io para criação, edição e gerenciamento de formulários dinâmicos.

---

## Objetivo do Projeto

Criar uma solução no-code que permita a construção de formulários personalizados, incluindo perguntas abertas e alternativas, além de visualizar respostas de modo simples e intuitivo.

---

## Funcionalidades

- Criar formulários com nome
- Adicionar perguntas com definição de tipo (texto ou alternativas)
- Criar opções de resposta para perguntas de múltipla escolha
- Exibir formulários para que outros usuários respondam
- Visualizar respostas individualmente em um painel dedicado
- Interface construída integralmente em Bubble.io

---

##  Prints da Aplicação

As imagens estão disponíveis na pasta `assets/` deste repositório:
- Tela de Login  
- Tela de Cadastro  
- Tela de Lista de Formulários: lista todos os formulários criados pelo usuário e permite editar, visualizar respostas, publicar e deletar
- Tela de Criação de Formulário: permite adicionar questões do tipo texto ou múltipla escolha
- Tela de Formulários Publicados: onde o usuário visualiza os formulários disponíveis e escolhe qual deseja responder
- Tela de Respostas: onde o usuário efetivamente responde ao formulário selecionado


## Ferramentas Utilizadas

- Bubble.io (desenvolvimento no-code)
- GitHub: documentação do projeto

## Aprendizados
