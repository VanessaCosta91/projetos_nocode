# Lista de Tarefas – Projeto No-Code em Bubble.io

Este é um projeto simples desenvolvido em Bubble.io para explorar a plataforma e entender como criar aplicações no-code.  
A aplicação permite cadastro de usuário, login, criação de tarefas, marcação como concluída.

---

## Objetivo do Projeto
Aprender o funcionamento básico do Bubble, incluindo:
- Criação de páginas e workflows  
- Uso de banco de dados interno  
- Autenticação de usuários  
- Criação e manipulação de listas (tarefas)

---

## Funcionalidades
- Cadastro e login de usuários
- Adicionar tarefas
- Marcar e desmarcar tarefas como concluídas
- Excluir tarefas
- Interface simples e funcional

---

## Prints da Aplicação
As imagens estão disponíveis na pasta `assets/` deste repositório:
- Tela de Login  
- Tela de Cadastro  
- Tela de Lista de Tarefas (marcar/desmarcar)

---

## Ferramentas Utilizadas
- Bubble.io (no-code)
- Banco de dados nativo da plataforma

---

## Aprendizados
Durante este projeto, explorei:
- Como estruturar workflows no Bubble  
- Como registrar e autenticar usuários  
- Como manipular dados simples (tarefas)  
- Lógica de criar e atualizar  
- Noções iniciais de construção de apps no-code
